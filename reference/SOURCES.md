# Filosofía

Siempre que exista una discrepancia entre varias fuentes:

1. La especificación tiene prioridad.
2. Después analizaremos la implementación de los navegadores.
3. Finalmente tendremos en cuenta la experiencia de profesionales reconocidos.

Nunca utilizaremos una única fuente para justificar una afirmación técnica cuando existan fuentes de mayor autoridad.
